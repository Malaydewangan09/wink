WINK(1) User Commands WINK(1)
name
wink - log aggregator for local dev services
synopsis
wink [command] [args...]
description
Watch any process on your machine. stdout and stderr from every
service, aggregated in a live terminal dashboard. Single binary,
zero config, no cloud.

commands
wink watch <name> <cmd>
Start a process and collect its output in the background.

wink attach <name> <pid>
Register an already-running process by PID.

wink ls
List all services and their current status.

wink logs [name]
Print logs, optionally filtered by service name.

wink tail [name]
Follow logs in real time without the TUI.

wink stop <name>
Send SIGTERM to a watched service.

wink clear
Remove all logs and state from ~/.wink/.
files
~/.wink/services.json
Registered services and their status.

~/.wink/logs.jsonl
Collected log lines, one JSON object per line.
