aggregated logs
stdout and stderr from all services in one scrollable view, color-coded by service.
log search
press / in the TUI to filter logs by keyword across any service in real time.
crash alerts
desktop notification when a service dies unexpectedly. configurable via notify_cmd.
wink up / down
define services in wink.yaml and start or stop them all with one command.
zero config
works with any process out of the box. optional config at ~/.wink/config.yaml.
single binary
no daemon, no dependencies, no setup. drop it in PATH and it works.
